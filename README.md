# Space-Invaders

### Description:

A game modified from the classic arcade space invaders.

![alt text](https://github.com/JoboFernandez/Space-Invaders/blob/master/imgs/selection_screen.png?raw=true)

Features include:
- Menu system
- Ship selection
- Weapon selection
- Round/Level system
- Enemy objects of varying behaviors
